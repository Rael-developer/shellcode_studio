{ShellCode_Studio}

Uma IDE Web Client-Side completa projetada para alta performance.

Desenvolvida para rodar diretamente no navegador, permitindo a renderização em tempo real de interfaces front-end (HTML/CSS/JS) e a compilação nativa de scripts Python via WebAssembly (Pyodide).

Guia Rápido

  • Ambiente Web: Escreva nos painéis HTML, CSS e JS. Clique em "Executar Web" para atualizar o Live Preview.
  • Ambiente Python: Escreva seu algoritmo e clique em "Executar Python". O Output será exibido no terminal integrado.
  • Download: Acesse a aba do código que deseja salvar e clique em "Baixar Código".
  • Atalho: Digite ! na aba HTML para gerar a estrutura básica.
